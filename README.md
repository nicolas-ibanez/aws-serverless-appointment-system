# AWS Serverless Appointment System

## Project Overview
This project is a scalable backend for an automated appointment management system. It leverages AWS Cloud technology to provide a high-performance, cost-effective solution for small businesses, such as barbershops or medical centers.

## Tech Stack
- **Compute:** AWS Lambda (Serverless Python)
- **Database:** Amazon DynamoDB (NoSQL)
- **Security:** AWS IAM with AdministratorAccess
- **Cost Management:** AWS Budgets with real-time alerting

## Project Roadmap 2026
- [x] AWS infrastructure & IAM security hardening
- [x] Cost control implementation ($120.00 USD credit managed)
- [ ] Core API development for appointment creation
- [ ] WhatsApp API & Calendly integration
